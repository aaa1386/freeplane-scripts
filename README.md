- همیشه آیکن سبز یا قرمز را ثابت باقی نگه می دارد

- بنابراین پنل تب ها شامل دو نوع تب می شود تب های مرتبط با نقشه کامل که همیشه آیکن و نام ثابت باقی می ماند نشانگر فایل ها است  و تب های زیر نقشه که آیکن قرمز دارد و ریشه های مختلف که انتخاب می شود نام تب  بروز می شود.
